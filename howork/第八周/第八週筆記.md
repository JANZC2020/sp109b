## pkg-config
* C/C++編譯器需要的輸入參數
* 連結器需要的輸入參數
* 已安裝軟體包的版本資訊
* pkg-config 是一個在原始碼編譯時查詢已安裝的函式庫的使用介面的電腦工具軟體。

## 指令
* ```sudo apt-get install libglib2.0-dev``` 安裝glib函式庫
* ```-lm``` 連結math.h這個函式庫，因為在c語言中math非標準函式庫，但c++則是
* ```apt install libsqlite3-dev``` 安裝sqlite3函式庫
* ```ps``` 查看背景及前景執行的程式
* ```&``` 背景執行
* ```#ifndef``` 如果沒定義則做後續
* ```define```定義某物
* ```endif``` 結束if
